## 📖 Sobre o projeto

O **Tesla Project** é um site inspirado no site oficial da Tesla, construído inteiramente com tecnologias de **frontend**. O objetivo do projeto é praticar e aprimorar habilidades em **HTML, CSS e JavaScript**, reproduzindo uma interface elegante, minimalista e responsiva.

Este projeto foi criado com fins **educacionais**, sem qualquer vínculo comercial com a Tesla, Inc.

---

## 🎯 Objetivos

- Praticar a construção de layouts modernos e responsivos.
- Aprimorar o uso de **Flexbox** e **Grid**.
- Trabalhar com animações e transições suaves.
- Aplicar boas práticas de semântica HTML e acessibilidade.
- Criar uma experiência de usuário fluida e atraente.

---

## 🚀 Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**

> Possíveis expansões futuras: integração com frameworks (React, Vue) e bibliotecas de animação (GSAP, AOS).
